# 元组

元组（tuple）把**固定数量、各不相同的类型**的值组合在一起。用圆括号 `( )` 创建，通过 `.0`、`.1` 等下标访问，或用 `let (a, b) = ...` 解构。

## 示例代码

源码在仓库根目录的 [`tuples.rs`](https://github.com/Huauauaa/hi-rust/blob/main/tuples.rs)：

<<< ../../tuples.rs{rust}

## 创建与访问

```rust
let point = (3, 4);
let mixed = ("Rust", 2024, true);
```

- 元组可以包含不同类型，例如 `(i32, i32)` 或 `(&str, i32, bool)`。
- 用 `.0`、`.1`、`.2` … 按**从 0 开始**的下标访问各元素。
- 元组是**定长**的：`(i32, i32)` 与 `(i32, i32, i32)` 是不同类型。

## 解构

用 `let` 模式匹配一次性取出各字段：

```rust
let (x, y) = point;
let (lang, year, stable) = mixed;
```

左侧变量个数和类型须与元组一致，否则编译不通过。

## 作为参数与返回值

元组可以传给函数，也可以从函数返回，适合一次返回多个值：

```rust
fn swap(pair: (i32, i32)) -> (i32, i32) {
    (pair.1, pair.0)
}
```

## 单元类型 `()`

空元组 `()` 是 Rust 的**单元类型**（unit type），表示「没有有意义的值」。无返回值的函数实际返回 `()`，例如 `fn foo() { }` 等价于 `fn foo() -> () { }`。

## 运行

在项目根目录执行：

```sh
rustc tuples.rs
./tuples
```

Windows 下运行 `tuples.exe`。

预期输出：

```text
=== 创建与访问 ===
point: (3, 4)
mixed: Rust, 2024, true
=== 解构 ===
x = 3, y = 4
lang = Rust, year = 2024, stable = true
=== 作为返回值 ===
swap(10, 20) = (20, 10)
=== 单元类型 () ===
空元组 () 是单元类型，表示没有有意义的值
```

## 小结

| 写法 | 说明 |
|------|------|
| `(a, b, c)` | 创建元组，元素类型可各不相同 |
| `t.0`、`t.1` | 按下标访问（从 0 起） |
| `let (x, y) = t` | 解构赋值 |
| `fn f() -> (T, U)` | 一次返回多个值 |
| `()` | 单元类型，表示无返回值 |
