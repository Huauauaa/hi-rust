# 条件判断

Rust 用 `if`、`else if`、`else` 做分支，用 `match` 按值模式匹配。`if` 的条件必须是 `bool`；`match` 必须覆盖所有可能值（或用 `_` 兜底）。两者都可以作为表达式返回值。

## if / else if / else

- 条件表达式必须是 `bool`。
- 每个分支用 `{}` 包裹；多行语句时花括号不能省略。
- `else if` 可以链式写多个分支，最后可选 `else` 兜底。

## if 作为表达式

`if` 分支的返回值类型必须一致，整体可以赋给变量：

```rust
let parity = if n % 2 == 0 { "偶数" } else { "奇数" };
```

单条语句时花括号可省略；多行语句时每个分支都要写 `{}`。

## 嵌套与组合条件

分支里可以再写 `if`。多个条件用 `&&`（与）、`||`（或）组合，与[运算符](/guide/operators)中的逻辑运算一致。

## match

`match` 把值与多个**模式**逐一比对，命中第一个分支后执行对应代码并结束。

- 每个分支写 `模式 => 表达式或块`。
- 多个模式可合并：`6 | 7 => ...`。
- 范围模式：`1..=5` 匹配闭区间内的整数。
- `_` 匹配任意值，常用于兜底；编译器要求所有情况都被覆盖。
- 整体是表达式，各分支返回值类型须一致。

```rust
let grade = match score {
    90..=100 => "A",
    80..=89 => "B",
    _ => "D",
};
```

## 小结

| 写法 | 说明 |
|------|------|
| `if cond { ... }` | 条件为真时执行 |
| `else if cond { ... }` | 前一个条件为假时再判断 |
| `else { ... }` | 以上都不满足时执行 |
| `if ... { a } else { b }` | 表达式，两分支类型须相同 |
| `match val { pat => ... }` | 按模式匹配，须穷尽所有情况 |
| `_` | 通配模式，匹配任意值 |
| `a \| b => ...` | 同一分支匹配多个模式 |
| `m..=n => ...` | 匹配整数闭区间 |
