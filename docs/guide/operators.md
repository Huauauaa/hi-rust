# 运算符

Rust 的运算符与多数 C 系语言类似，但整数除法会截断小数，比较与逻辑运算的结果是 `bool`。

## 示例代码

源码在仓库根目录的 [`operators.rs`](https://github.com/Huauauaa/hi-rust/blob/main/operators.rs)：

<<< ../../operators.rs{rust}

## 算术

| 运算符 | 说明 |
|--------|------|
| `+` | 加法 |
| `-` | 减法 |
| `*` | 乘法 |
| `/` | 除法（整数相除截断小数） |
| `%` | 取余 |

整数类型之间可以混用需显式转换；浮点数用 `f32` / `f64`。

## 比较

| 运算符 | 说明 |
|--------|------|
| `==` | 相等 |
| `!=` | 不等 |
| `<` `>` `<=` `>=` | 大小比较 |

比较结果是 `bool`，常用于 `if` 和 `match`。

## 逻辑

| 运算符 | 说明 |
|--------|------|
| `&&` | 与（短路） |
| `\|\|` | 或（短路） |
| `!` | 非 |

`&&` 和 `||` 会短路：左侧已能决定结果时不求值右侧。

## 位运算

对整数类型按二进制位操作，常用于标志位、掩码：

| 运算符 | 说明 |
|--------|------|
| `&` | 按位与 |
| `\|` | 按位或 |
| `^` | 按位异或 |
| `!` | 按位取反 |
| `<<` | 左移 |
| `>>` | 右移 |

## 复合赋值

在算术、位运算结果上写回变量：`+=`、`-=`、`*=`、`/=`、`%=`，以及 `&=`、`|=`、`^=`、`<<=`、`>>=`。

## 运行

在项目根目录执行：

```sh
rustc operators.rs
./operators
```

Windows 下运行 `operators.exe`。

预期输出：

```text
=== 算术 ===
a + b = 13
a - b = 7
a * b = 30
a / b = 3
a % b = 1
=== 比较 ===
a == b: false
a != b: true
a > b: true
a < b: false
a >= b: true
a <= b: false
=== 逻辑 ===
t && f: false
t || f: true
!t: false
=== 位运算 ===
x & y = 00001000
x | y = 00001110
x ^ y = 00000110
!x    = 11110101
x << 1 = 00010100
x >> 1 = 00000101
=== 复合赋值 ===
n += 2 → 7
n -= 1 → 6
n *= 3 → 18
n /= 2 → 9
n %= 4 → 1
```

## 小结

| 类别 | 典型运算符 | 结果类型 |
|------|-----------|----------|
| 算术 | `+` `-` `*` `/` `%` | 与操作数相同（或推断） |
| 比较 | `==` `!=` `<` `>` | `bool` |
| 逻辑 | `&&` `\|\|` `!` | `bool` |
| 位运算 | `&` `\|` `^` `!` `<<` `>>` | 整数类型 |
